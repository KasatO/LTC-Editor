1.0.10 RC3 - 24.09.2009
+ Доработана загрузка изменений из файлов от FM 2009 и старше;
* Исправлена ошибка потери последней строки при отсутствии в файле склонений.

1.0.10 RC2 - 11.11.2008
+ Добавлена поддержка FM 2009 (LTC и LTF файлы новых версий);
+ Добавлена поддержка склонений (в LTC и LTF файлах);
+ Добавлен расширенный поиск и замена;
+ Добавлена возможность отключения проверки орфографии (временно она отключена совсем);
+ Изменена версия файлов работы, старые файлы будут несовместимы с новой версией программы;
+ Переход по флагам (маркерам строк) переделан на переход по блокам флагов;
+ Полная поддержка Unicode (включая интерфейс, работу с различными локалями);
* Небольшие изменения в интерфейсе.

1.0.9 - 20.10.2006
+ Добавлена проверка орфографии русского и английского языка (язык выбирается принудительно по состоянию непереведенности строки) при помощи MS Word, который загружается в фоне и используется для приема, проверки и отправки обратно текста;
+ Добавлена подсветка в поле редактирования тагов, комментариев, переменных, знаков препинания и искомой строки (если что-то в ней набрано);
+ Добавлен режим загрузки изменений более медленным и точным способом с перебором всех строк и в любом порядке;
+ Добавлено создание отчета о загруженных изменениях: в каталоге программы создается файл log.htm с информацией, какая строка как себя "повела";
- Исправлена иконка, присваемая файлам при регистрации расширений *.ltc, *.lrf, *.lew под программу;
* Изменен алгоритм применения изменений и отображения состояний строки;
- Исправлены некоторые другие глюки и недочеты.

- Исправлена ошибка неперехода к следующей строке при нажатии Enter в поле редактирования;
* Отслеживание статуса переводимой строки (непереведена, ошибочна и т.д.) происходит и во время изменения текста в поле редактирования;

+ Для регистрации в реестре доступно и "родное" расширение *.lew.

1.0.6 - ?
+ Добавлен подсчет и отображение информации о количестве "маркированных" строк;
+ Добавлена возможность открытия файлов перевода (*.ltc и *.ltf) программой как обычных документов - двойным кликом (регистрация и удаление регистрации этих типов под программу);

- Исправлен баг при сохранении файла, в котором присутствовали строки с несколькими вариантами перевода, приводивший к "размножению" этих строк.

1.0.5 - 21.02.2006
+ Кнопки поиска (F3 и Enter) теперь работают и тогда, когда в фокусе основной спискок;
+ Добавлена опция, включающая и выключающая массовое выделение строк в списке; рекомендую включать его только в случае необходимости, и после массовых правок отключать, т.к. при его включении поиск и редактирование может глючить (выделяется несколько строк, что сложно отследить визуально, поэтому правка может примениться не к желаемой строке); опция не сохраняется в реестре;
+ Добавлен экспорт в *.ltf-файлы, совместимые с FM2005 и бета-версиями FM 2006 (без сохранения падежей);

1.0.4 - 17.02.2006
+ Добавлено сохранение строк из контекстного меню (сперва нужно выбрать строки);
+ Добавлена массовая "обработка" (удаление и восстановление) строк;
+ Добавлена подсветка фона поля для редактирования, аналогичная подсветке фонов строк в основном списке;
* Ускорена загрузка изменений засчет уменьшения "точности" - поиск строк стал последовательным (последующая ищется с предыдущей найденной), потому как при сохранении их порядок должен был быть такой же;
* При загрузке изменений стрОки, которые реально не изменились, теперь не выделяются, как измененные;
+ Добавлено сохранение оригинальных строк (переводов) при загрузке изменений, поэтому теперь их можно восстанавливать;
+ При наведении на индекс отображается расширенная подсказка с реальным (ФМ-овским) индексом и состоянием всех флагов;
- Исправлен недочет, когда при загрузке и применении изменений не сбрасывался флаг непереведенной строки;
- Исправлены мелкие ошибки.

1.0.3 - 10.02.2006
+ Сохранять теперь можно не только измененные строки, но и любой набор разных "маркированных" строк и их сочетаний, а также любой диапазон строк (с нумерацией по индексам);
+ Добавлена возможность сохранять и загружать "работу" - набор переменных, списков, проделанных изменений, названий открытых файлов; "работа" имеет расширение .lew и занимает порядка 50-60 Мб для русского языка;
+ Добавлена поддержка английского языка (и вообще мультиязычность - надо просто создать файл *.lng и положить его в каталог с программой, она сама все найдет);
+ Доступно восстановление измененных вручную строк (через контекстное меню);
+ Колонка списка, в которой отображается перевод, теперь меняет название в соответсвии с загруженным файлом;
+ Добавлено запоминание некоторых настроек - размеров и положения окна и выбранного языка;
* При открытии файла в название окна добавляется только имя файла, а не имя и путь вместе;
* Поддержка WinXP встроена в программу, поэтому внешний manifest убран;
+ Перед открытием файла происходит проверка на изменения, если они присутсвуют - будет выдан запрос на запись;
- Исправлены найденные глюки.

1.0.2 - 07.02.2006
+ Добавлена возможность удаления (и восстановления) строк через контекстное меню списка, появилась новая подсветка - серая;
+ В контекстном меню также добавлена опция вставки (дублированием) строк, имеющих тот же идентификатор, что и оригинальная строка, и работающих как новый вариант перевода (ФМ поддерживает такую возможность); подсветка - желтая;
+ Добавлены опции отключения автопоиска при вводе и игнорирования регистра вводимых симолов;
+ Добавлен экспорт перевода версии v.4 в файл версии v.3;
+ Добавлена возможность изменения размера области редактирования (полей английского текста и перевода);
* Немного изменен алгоритм определения вероятной ошибки - выделяются ОБЕ вероятно ошибочные строки в случаях, когда индексы непоследовательны, либо индексы равны между собой или одинаков английский текст И одинаков перевод.
